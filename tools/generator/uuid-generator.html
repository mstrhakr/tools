    UUID Generator

    Generate random v4 UUIDs or deterministic v5 UUIDs from namespace and name

      How it works

      UUIDs (Universally Unique Identifiers) are 128-bit values standardized in RFC 9562. Version 4 fills 122 bits with cryptographically random data; version 5 hashes namespace + name with SHA-1 and encodes deterministic identifiers.

      This tool uses the browser's native crypto.randomUUID() — no external library, no server call. The output is formatted as xxxxxxxx-xxxx-4xxx-yxxx-xxxxxxxxxxxx where 4 indicates v4 and y is one of 8, 9, a, or b (variant bits).

      Use v4 for random IDs and v5 when the same input should always map to the same identifier (for example deriving stable IDs from domains or paths).
