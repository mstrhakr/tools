Regex help for beginners
TokenWhat it means
^Start of text (or line with m flag)
$End of text (or line with m flag)
[abc]One character from this list
[a-z]One lowercase letter from a to z
[^a-z]One character that is not a-z
\dOne digit (0-9)
\wWord character (letters, numbers, underscore)
\sWhitespace (space, tab, newline)
+One or more
*Zero or more
?Zero or one
{m,n}Between m and n times
(...)Capture group
|Or

Tip: keep patterns strict when validating user input. Use ^ and $ so partial matches do not pass by accident.
