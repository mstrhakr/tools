    Supported range syntax

    • Comparators: >, >=, <, <=, =
    • Tilde: ~1.2.3
    • Caret: ^1.2.3
    • Wildcards: 1.x, 1.2.*
    • Space-separated AND clauses, for example >=1.2.0 <2.0.0
