mstrhakr tools

    MIME Type Lookup

    Look up a MIME type by file extension or reverse-search common extensions for a content type

    How to use it

    • Enter an extension like json or .tar.gz to look up the most common content type.
    • Enter a MIME type like application/json to see common extensions and a short note.
    • The dataset is local and intentionally focused on common engineering and web file types rather than every registered media type.
