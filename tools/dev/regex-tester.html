Regex help for beginners
TokenMeaningExample
^ and $Start and end anchors. Use both to force a full match.^\\d+$
[abc]One character from this set.[aeiou]
[a-z]One character in this range.[A-Z]{2}
\\d / \\w / \\sDigit / word char / whitespace shortcuts.\\w+@\\w+\\.\\w+
+One or more.\\d+
*Zero or more.a*
?Optional (zero or one).colou?r
{m,n}Repeat between m and n times.[a-z]{3,8}
(...)Capture group (shown below in match results).(\\d{4})-(\\d{2})
|OR between alternatives.cat|dog

Quick tip: start simple, test often, and only add complexity one token at a time.
