    How This Works

    This tool uses the same backend GeoIP endpoint as the GeoIP Lookup tool. When you click "Lookup My IP," the backend reads your public address from the HTTP request's source IP — the same way every web server knows who's connecting. No JavaScript geolocation API, browser permissions, or tracking cookies are involved.

    The IP shown is your public IP as seen by the internet — the address your ISP assigns to your router (or VPN exit node). All devices on your home network share this single address via NAT. Mobile carriers using CGNAT may share one public IP among thousands of subscribers.

    • IPv4 vs IPv6 — if your ISP and browser support IPv6 and prefer it, the tool shows your IPv6 address. Both are valid public addresses.
    • VPN / proxy — the IP shown will be the VPN server's exit address, not your ISP address.
    • ASN — identifies the network operator. Residential ISPs, cloud providers, and CDNs each have distinct ASN ranges and are easy to distinguish.
