    Why RDAP

    RDAP (Registration Data Access Protocol) is the modern replacement for WHOIS. It returns JSON instead of plain text, so key fields like owner, status, nameservers, and event dates can be parsed reliably.

    • Domains typically return registrar and nameserver data.
    • IP lookups return allocation range, network name, and RIR ownership details.
    • Contact data may be redacted depending on registry policy and GDPR rules.
