    How This Works

    Traditional WHOIS uses a plain-text TCP protocol (port 43) — not something a browser can connect to. The backend queries the appropriate registry using RDAP (Registration Data Access Protocol), the modern JSON-based WHOIS successor standardized by ICANN. It returns structured, parseable data rather than free-form text.

    The lookup is two-step: the IANA bootstrap registry identifies which regional authority manages the TLD or IP block, then that authority is queried directly. For example, .com goes to Verisign, IP addresses go to ARIN (North America), RIPE (Europe), APNIC (Asia-Pacific), LACNIC (Latin America), or AFRINIC (Africa).

    • Nameservers — the authoritative DNS servers for this domain. Changes propagate globally in 24–48 hours.
