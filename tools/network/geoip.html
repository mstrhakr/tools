    How This Works

    The backend queries a locally hosted MaxMind GeoLite2 database — a periodically updated dataset mapping IP ranges to geographic and organizational data. No third-party API call is made per lookup; the database is bundled server-side for fast, private results.

    Accuracy varies significantly by data type:

    • Country — ~99% accurate. IP registrations are organized by country and rarely cross borders.
    • City — ~80% accurate to within 50km. ISPs often assign IP blocks from a regional hub, not the user's actual city.
    • Street-level — not reliable. GeoIP is not a GPS lookup; IP blocks are assigned to data centers or exchange points, not street addresses.
    • ASN / ISP — very accurate. Each Autonomous System Number (ASN) is publicly registered with the owning organization.

    VPNs, Tor exit nodes, and cloud provider IPs (AWS, Azure, Google Cloud) will show the data center location and organization, not the end user's actual location.
