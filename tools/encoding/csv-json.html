    CSV ↔ JSON Converter

    Convert between CSV and JSON. First row of CSV is treated as headers.

    How it works

    CSV (Comma-Separated Values) and JSON are both common data interchange formats. This converter parses one and produces the other entirely in your browser — no data is sent anywhere.

    CSV → JSON: The first row is treated as column headers. Each subsequent row becomes a JSON object with those keys. Values are kept as strings unless they parse cleanly as numbers.

    JSON → CSV: Expects an array of objects at the top level. Keys from the first object become headers; missing keys in later rows produce empty cells.

    delimiter note: CSV has no single formal spec (RFC 4180 is informational). Spreadsheet exports often use semicolons in European locales and tabs in TSV files — use the delimiter selector to match your data. Fields containing the delimiter character must be wrapped in double quotes, which this parser handles automatically.
