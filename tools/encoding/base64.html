mstrhakr tools

    Base64 Encode / Decode

    Encode and decode Base64 strings with full UTF-8 support

    How it works

    Base64 encodes arbitrary binary data as a sequence of 64 printable ASCII characters: A–Z, a–z, 0–9, +, and /, with = as padding. Every 3 bytes of input become 4 characters of output, resulting in ~33% size overhead.

    It was designed to transmit binary data over channels that only handle plain text — email attachments (MIME), HTTP Basic Auth headers, and CSS data URIs all use Base64. It is encoding, not encryption: anyone can decode it instantly.

    This tool handles full UTF-8 text correctly by passing input through TextEncoder before encoding, and through TextDecoder after decoding — so emoji, accented characters, and non-Latin scripts all work as expected.
