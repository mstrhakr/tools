mstrhakr tools

    XML Formatter

    Pretty-print and validate XML documents client-side with no network requests

    Formatting notes

    • The parser validates for well-formed XML, including matching tags and correct nesting.
    • Formatting normalizes indentation while preserving element text, attributes, comments, and processing instructions.
    • This is an XML formatter, not an XSD validator. Schema validation belongs in a separate tool.
