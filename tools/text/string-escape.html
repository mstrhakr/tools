    String Escape / Unescape

    Escape or decode strings for JSON, JavaScript, HTML, and URLs without leaving the browser

    Formats supported

    • JSON / JavaScript string turns control characters into sequences like \n and \t.
    • HTML entities is useful when you need literal markup characters rendered safely in a page or snippet.
    • URL encoding applies percent encoding, which is appropriate for query parameter values and path components.
