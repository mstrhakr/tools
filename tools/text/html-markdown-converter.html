    HTML / Markdown Converter

    Convert simplified Markdown to HTML and convert common HTML tags back to Markdown

    Scope

    • The converter supports the most common formatting patterns: headings, emphasis, links, lists, code blocks, and paragraphs.
    • It is intentionally lightweight and deterministic for quick transformations in the browser.
    • For strict CommonMark and full HTML fidelity, use a dedicated parser chain in your build pipeline.
