    How it works

    All date math is done using JavaScript's built-in Date object, which internally stores time as milliseconds since the Unix epoch: January 1, 1970, 00:00:00 UTC. Differences are computed in milliseconds and then broken down into days, hours, minutes, and seconds.

    The epoch date was chosen as the approximate birth of Unix at Bell Labs. Adding/subtracting durations is handled carefully for calendar irregularities (variable month lengths, leap years) using the Date API rather than raw arithmetic.

    Gotcha to watch out for: months in JavaScript's Date API are zero-indexed (January = 0, December = 11), but the display and input are 1-indexed as you'd expect. This tool handles that transparently.
