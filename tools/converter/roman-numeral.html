    Roman Numeral Converter

    Convert between decimal numbers and Roman numerals with canonical validation

    Rules used

    • The standard range is 1 through 3999 using the symbols I, V, X, L, C, D, and M.
    • Subtractive notation is supported for values such as IV, IX, XL, XC, CD, and CM.
    • Roman numeral input is re-encoded after parsing so non-canonical forms can be rejected.
