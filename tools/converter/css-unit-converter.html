    CSS unit primer

    CSS has two categories of length units: absolute (fixed physical sizes) and relative (relative to something else — font size, viewport, parent element).

    • px — CSS pixels. On standard screens, 1 CSS px = 1 physical pixel. On HiDPI/Retina displays, 1 CSS px = 2 or more physical pixels (device pixel ratio). Everything converts through px as the base unit.
    • rem — relative to the root element (<html>) font size. The browser default is 16px. Great for scalable, accessible design: users who change their browser font size get proportionally sized UI.
    • em — relative to the current element's font size (or its parent's, for font-size itself). Beware of compounding: an element with em inside another with em will multiply.
    • pt / pc — print units. 1pt = 1/72 inch; 1pc (pica) = 12pt. Rarely used in web CSS except when targeting print stylesheets.
    • vw / vh — 1% of viewport width / height. Useful for fluid layouts and full-screen sections.
