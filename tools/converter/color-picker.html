    How it works & color model primer

    All three color formats represent the same value — they're just different ways to express it. Conversions are pure math done in your browser.

    • Hex (#RRGGBB) — Red, Green, Blue each as a two-digit hexadecimal number (00–FF). The most common format for web colors. Shorthand like #RGB expands each digit to two identical digits (#abc = #aabbcc).
    • RGB (0–255, 0–255, 0–255) — hardware-oriented; mirrors how monitors work by mixing red, green, and blue light. Each channel is an 8-bit integer. Black = 0,0,0 (no light); white = 255,255,255 (full of all three).
    • HSL (Hue, Saturation, Lightness) — human-oriented. Hue is a 0–360° angle around the color wheel (0° = red, 120° = green, 240° = blue). Saturation is how vivid vs. grey (0–100%). Lightness controls brightness (0% = black, 50% = full color, 100% = white). Much more intuitive for design adjustments.
